Conhecimentos de HTML

Durante as aulas de autoria web,aprendemos sobre html,uma ferramenta muito interessante na progamação de sites,e com isso,irei citar algumas coisas que aprendemos nas aulas com essa ferramenta

Comandos básicos

Vamos aprender os comandos básicos:

    p - Parágrafo
    H1 até H6 - Títulos,quanto maior o núemro menor o título
    title - O título da página
    br - Quebra de linha
    head - a "cabeça" do código,onde você pode colocar códigos que não quer que sejam mostrados
    body - O "corpo" do código,onde você pode colocar os conteúdos principais que querem ser mostrados
    img src - Usado para colocar imagens
    a href - Geralmente usado para colocar links

CSS

Além dos códigos,temos o css,que também ajuda o código html de certa forma,ele serve para estilizar o site e deixar vizualmente mais bonito e atrativo, o css tem 3 tipos:

    Inline - Estilização feita na própria linha do código
    Interno - Estilização feita dentro do código html
    Externo - Estilização feita em um local separado porém ligada ao código html
